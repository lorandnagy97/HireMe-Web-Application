Javascript Procedural Game


This game utilizes pure javascript, with no frameworks, to draw onto a HTML5 canvas element. The map is generated via chunks, and each chunk generates its own objects. The purpose of this is to both showcase my current abilities in Javascript, as well as to show my ability to work with lower levels of abstraction. There's not much to it yet, but the source code is available upon request.
